## üìñ Overview

This repository contains a comprehensive Exploratory Data Analysis (EDA) of a global air quality dataset. The project aims to uncover insights into air pollution patterns, identify relationships between different pollutants, and explore the influence of meteorological factors on air quality across various cities worldwide.

The analysis is presented in a Jupyter Notebook (`EDA.ipynb`) that walks through the data cleaning, processing, visualization, and interpretation steps.

## üìä Dataset

The dataset used in this project is the "Global Air Quality Dataset" from Kaggle. It contains 10,000 records of air quality measurements from 27 cities across 21 countries.

- **Source:** [Global Air Quality Dataset on Kaggle](https://www.kaggle.com/datasets/waqi786/global-air-quality-dataset)
- **Uploaded by:** [Waqar Ali](https://www.kaggle.com/waqi786)

### Features

The dataset includes the following columns:

- `City`: Name of the city
- `Country`: Name of the country
- `Date`: Date of the measurement
- `PM2.5`: Particulate Matter 2.5 concentration (in ¬µg/m¬≥)
- `PM10`: Particulate Matter 10 concentration (in ¬µg/m¬≥)
- `NO2`: Nitrogen Dioxide concentration (in ppb)
- `SO2`: Sulfur Dioxide concentration (in ppb)
- `CO`: Carbon Monoxide concentration (in ppm)
- `O3`: Ozone concentration (in ppb)
- `Temperature`: Temperature (in ¬∞C)
- `Humidity`: Humidity (in %)
- `Wind Speed`: Wind speed (in km/h)
